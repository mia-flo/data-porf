About Me

  • Resume

I am a geoscientist turned data analyst in San Francisco, CA.

As a PhD candidate in the Geological Sciences at Stanford University, I spent 4 years collecting, analyzing, visualizing, and interpreting geochemical data to better understand rates, magnitude, and processes of geological phenomena.

Now, I am expanding my knowledge with both data analysis and machine learning projects that utilize Python and SQL . My projects so far range from cleaning data and exploratory analysis to building machine learning models for predictive modelling.

I have a lifelong passion for learning and bring a diverse skill set, communication skills, as well as experience making data-driven decisions.

My current learning goals: improve in statistics and mathematics for both data analysis and machine learning

Current personal project: using Gemini AI to extract data from my grocery receipts in order to build a database and an interactive Tableau dashboard that tracks my household's grocery costs.

Outside of work, my recent hobbies are indoor bouldering, learning Japanese, and playing Slay the Spire 2.

Check out the project page or my GitHub to learn more about some of my personal projects.
