Utilizing Gemini AI to extract product review data and perform simple sentiment analysis in Google Sheets

Data Analysis

AI

Objective: To explore the use of AI in a data analysis workflow for a project focused on analyzing text-based data.

Results: So far I have found to be most useful for creating summaries such as pivot tables and analyzing text.

Data set: sourced from REI's La Sportiva TC Pro Climbing Shoes

See more projects on my the project page or my GitHub or scroll down for a more in-depth look!

Tasks I gave to Gemini:

  • To extract review data into a table
  • Simple sentiment analysis of reviews (Positive, Neutral, Negative)
  • 1-word categorization (Sizing, Material, Durability, Comfortability, Price, Performance) of the primary like or dislike extracted from reviews
  • Recommend data normalization techniques for the 'Thumbs up' and 'Thumbs down' count on individual reviews to assess engagement and the helpfulness of a review
  • Recommend sample weighting techniques in order to quantify the importance of each review based on engagement
  • Create a pivot table that summarizes the data by grouping reviews by sentiment, complaint, star rating, and level of engagement
  • Create a summary tab targeted towards executives

Pros of using Gemini in the workflow

  • Data extraction was easy and fast.
  • Simple sentiment analysis was quick, easy, and generally accurate.
  • Categorization, when provided with enough categories, was mostly accurate.
  • Is able to explain some pros and cons when recommending techniques or calculations.
  • Gemini generated a decent summary of the data.
  • Generating the pivot table was fast and easy - it even asked clarifying questions.
  • The executive summary tab I asked it to create was generated a bit faster than if done by hand.

Cons when using Gemini in the work flow

  • Data point extraction was limited in number (15 out of 80+ reviews). Scraping the webpage manually would have resulted in a much richer data set.
  • The information given by Gemini wasn't always entirely accurate - an example shown in the figure above: it said 29 votes was the highest engagement when it was in fact 73.
  • Certain tasks are completed more efficiently by hand ex: conditional formatting and some simple plots or calculations.
